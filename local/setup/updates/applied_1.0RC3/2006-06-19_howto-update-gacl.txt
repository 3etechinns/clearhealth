How-to add 'override' ACO 

* Go to Admin > Setup > ACL Editor
* Select "Actions" ACO Section
* Click on "Edit" link under Access Control Objects
* In "Add ACOs" add the following value/order/name: override/1/override
* Click Submit
* Click "ACL List"
* Click 'Edit' on the first ACL with the following note:
	"Give Superadmn and access to everything even when no resource is selected"
* Select "Actions" in ACO Section
* Select "override" in Access Control Objects
* Click the >> button to add ACO
* Click "Submit"
