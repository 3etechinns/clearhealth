Category File Format Info

Standard Format

Basic Format Category Name Newline
Basic + Category Id Category Id-Category Name Newline
Category Category Id-Category Name Whitespace Code Newline
Code Range Category Whitespace Code Start-Code End Newline

If no category Id is specified (or the given id already exists) one will be automatically generated

For example:
Category One
2-Category Two
Category Three			1100
4-Category Four			500-1000

Code to Category Mapping format

An alternate format is supported for importing code to category mappings (uses the Category Id). To enable this format put CODEMAP as the first line.
Code Whitespace Category Id Newline
For example:

CODEMAP
99214		1
99202		2
